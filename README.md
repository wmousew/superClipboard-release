## 🚀 安装使用

### 系统要求

- **操作系统**: Windows 10 / Windows 11（64位）
- **运行库**: 已内置 MSVC 运行时，无需额外安装
- **存储空间**: 约 120 MB 解压后

### 安装步骤

1. 下载 `superClipboard-release.zip`
2. 解压到任意目录（如 `D:\tools\superClipboard`）
3. 双击运行 `superClipboard.exe`

> 💡 **提示**：建议将程序固定到任务栏，方便随时使用全局快捷键 `Ctrl + Space` 唤出。
